# UI/UX Design Prompt – KrishiPath Company Dashboard (MVP)

## Objective

Design a premium, modern, and production-ready **Company Dashboard** for **KrishiPath**, an agricultural engagement platform where pesticide, fertilizer, seed, bio-fertilizer, and agri-input companies can educate farmers, reward them for engagement, and receive high-intent callback requests.

This is **not** a traditional advertising platform like Meta Ads or Google Ads.

Instead, companies create **Learning Campaigns** that help farmers understand products through educational videos, brochures, and quizzes while rewarding them for participation. Farmers can also request a callback if they are interested in learning more about a product.

The final design should feel like a premium SaaS application that a real agricultural company would confidently use from day one.

---

# Design Goals

The UI should communicate:

* Transparency
* Simplicity
* Trust
* Professionalism
* Premium quality
* Agriculture-first experience
* Data-driven decision making

The interface should be clean, intuitive, and require almost no training.

Every screen should naturally answer one question:

> **"What should the company do next?"**

---

# Design Style

Create a modern SaaS dashboard using the existing **KrishiPath** design language.

Requirements:

* Match KrishiPath's existing green color palette.
* Maintain visual consistency with the current application.
* Premium and elegant appearance.
* Minimal and uncluttered layouts.
* Large spacing.
* Rounded cards.
* Soft shadows.
* Subtle gradients.
* Glassmorphism only where it improves the design.
* Beautiful data visualization.
* Excellent typography.
* Smooth micro-interactions.
* Responsive desktop-first design.
* Mobile adaptation should also be considered.
* Follow modern UX best practices.

Avoid generic admin panel designs.

---

# Campaign Builder (Most Important Screen)

Design a guided campaign creation wizard.

### Step 1 – Campaign Information

* Campaign Name
* Campaign Goal
* Short Description

### Step 2 – Upload Content

* Upload Video
* Upload PDF Brochure
* Create Quiz
* Quiz Questions
* Quiz Answers
* Correct Answer Selection

### Step 3 – Reward Configuration

* Video Reward
* Quiz Reward
* Brochure Reward
* Callback Reward
* Daily Budget
* Campaign End Date

Show a live cost estimator that updates automatically.

Example:

* 500 Video Views
* 300 Quiz Completions
* 200 Brochure Downloads
* 20 Callback Requests

Display the estimated total reward distribution and platform charges before publishing.

### Step 4 – Target Farmers

Allow targeting by:

* State
* District
* Crop
* Preferred Language

Future filters should appear disabled with a "Coming Soon" badge.

### Step 5 – Review & Publish

Display:

* Campaign Summary
* Estimated Reach
* Estimated Reward Cost
* Platform Fee
* Wallet Balance After Launch

Primary CTA:

**Launch Campaign**

The complete process should take less than five minutes.

---

# UX Requirements

* The first campaign should be created in under 5 minutes.
* Minimize clicks wherever possible.
* Every page should clearly guide the next action.
* Prioritize readability and simplicity.
* Ensure accessibility with strong color contrast and clear typography.
* Design for scalability so additional features can be added later without redesigning the interface.

---

# Final Goal

Create an MVP that feels polished, trustworthy, and enterprise-ready.

The dashboard should convince companies that:

* Their wallet and spending are fully transparent.
* Campaign creation is effortless.
* Farmer engagement is measurable.
* Callback requests are easy to manage.
* KrishiPath is a reliable long-term marketing platform.

The final result should look like a premium product comparable to modern SaaS platforms such as Stripe Dashboard, Notion, Linear, or HubSpot, while maintaining a distinct agricultural identity aligned with the KrishiPath brand.
